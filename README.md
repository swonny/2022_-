    - a : a가 가리키는 곳의 주소 (포인터변수 b의 주소)

    - **a : a가 가리키는 곳(b)이 가리키는 곳(C)이 담고있는 값

    - b : b가 가리키는 곳의 주소(C의 주소)
    - *b : b가 가리키는 곳의 값 (C의 값)

    **head -> *newNode -> struct Node

    - (*head) : head가 가리키는 곳이 가리키는 값
